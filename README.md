# Northline Studio Hub

A lightweight internal web app for collecting and organizing ideas, schedules, and references in one place.

## Open the app

Open this file in your browser:

- `C:\Users\alewa\OneDrive\Documents\Virtual Whiteboard\index.html`

No build step or server is required.

## What it supports

- Quick capture for emails, call notes, screenshots, docs links, and studio ideas
- Board workflow across Inbox, Planned, Designing, Client Review, and Done
- Schedule view for due dates and unscheduled work
- Library view grouped by source
- Team ownership for 8 members
- Search and filtering by source and owner
- Local persistence in browser storage
